Установка

Вставьте код на ваш сайт в место, где должна располагаться кнопка "Оплатить":


    <div id = "qr-container"></div>
    <form name = "iceBitsyPayForm" onsubmit="iceDevApi.qrCodeSubmit(); return false;">
        <input class = "iceBitsyPayRow" type = "hidden" name = "shopAuthCode" value = "MyshopAuthCode">
        <input class = "iceBitsyPayRow" type = "hidden" name = "code" value = "MyCode">
        <input class = "iceBitsyPayRow" type = "hidden" name = "currency" value = "RUB">
        <input class = "iceBitsyPayRow" type = "text" name = "amount" placeholder = "Сумма заказа" value = "" required>
        <input class = "iceBitsyPayRow" type = "text" name = "order" placeholder = "Номер заказа" value = "">
        <input class = "iceBitsyPayRow" type = "text" name = "description" placeholder = "Описание заказа" value = "">
        <input class = "iceBitsyPayRow" type = "text" name = "name" placeholder = "ФИО плательщика" value = "">
        <input class = "iceBitsyPayRow" type = "text" name = "email" placeholder = "E-mail" value = "">
        <input class = "iceBitsyPayRow" type = "text" name = "phone" placeholder = "Контактный телефон" value = "">
        <input class = "iceBitsyPayRow" type = "submit" name = "payButton" value = "Оплатить">
    <form/>

Сумма заказа указывается в рублях.

Протестировать виджет можно здесь.

Настройка

Получите идентификатор авторизации пользователя и код магазина в разделе «web - терминалы» мобильного приложения.Вставьте значения атрибута value поля shopAuthCode и code:


    <input class = "iceBitsyPayRow" type = "hidden" name = "shopAuthCode" value = "MyshopAuthCode">
    <input class = "iceBitsyPayRow" type = "hidden" name = "code" value = "MyCode">

Для установки суммы платежа заполните значение атрибута value поля amount


    <input class = "iceBitsyPayRow" type = "text" name = "amount" placeholder = "Сумма заказа" value = "" required>

Для установки остальных полей, при необходимости, заполните значение атрибута value соответствующего поля 


        <input class = "iceBitsyPayRow" type = "hidden" name = "currency" value = "RUB">
        <input class = "iceBitsyPayRow" type = "text" name = "order" placeholder = "Номер заказа" value = "">
        <input class = "iceBitsyPayRow" type = "text" name = "description" placeholder = "Описание заказа" value = "">
        <input class = "iceBitsyPayRow" type = "text" name = "name" placeholder = "ФИО плательщика" value = "">
        <input class = "iceBitsyPayRow" type = "text" name = "email" placeholder = "E-mail" value = "">
        <input class = "iceBitsyPayRow" type = "text" name = "phone" placeholder = "Контактный телефон" value = "">

Чтобы скрыть поле платежной формы, добавьте атрибут type со значением hidden:


        <input class = "iceBitsyPayRow" type = "hidden" name = "phone" placeholder = "Контактный телефон" value = "">

Чтобы сделать поле платежной формы обязательным для заполнения, добавьте атрибут required:


    <input class = "iceBitsyPayRow" type = "text" name = "amount" placeholder = "Сумма заказа" value = "" required>

Получение кода

Для получения идентификаторов web-терминала необходимо:

Скачать и установить приложение "ice-qr".

Пройти процедуру регистрации в приложении.

перейти на вкладку "web-терминалы".

Нажать кнопку "Добавить терминал".

Ввести данные web-терминала и нажать кнопку "Сохранить". После регистрации web-терминала появяться идентификационные данные.

Их можно отравить нажав на кнопку "Отправить данные"
